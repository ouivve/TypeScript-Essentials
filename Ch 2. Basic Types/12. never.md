# 2.12 never

never 타입은 모든 타입의 subtype이며, 모든 타입에 할당할 수 있다.

하지만 never에는 그 어떤 타입도 할당할 수 없다.

- any 조차도 never에 할당할 수 없다.

잘못된 타입을 넣는 실수를 막고자 할 때 사용하기도 한다.

```typescript
let a: string = "hello";

if (typeof a! == 'string') {
    let b: never = a;
}

typeIndexable<T> = T extends string? T & {[index: string]: any} : never;
```
